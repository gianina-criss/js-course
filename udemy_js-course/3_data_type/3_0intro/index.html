Primitive Type

  • String - sequence of characters, must be in quotes or backticks. 
    "string"
  • Number - integers, pozitiv or negativ numbers as well as floating-point numbers. 
    2
  • Boolean - logical entity / true or false.
  • Null - intentional absence of any object value.
  • Undefined - a variable that has not been defined / assigned.
  • Symbol - built-in object whose constructor returns a unique symbol.
  • Biglnt - big integers greater that the 'number' type can handle.

Reference Type (Objects)

  • Are a non-primitive value and when assigned to a variable, the variable is giving a reference to the value.
  • Objects literals, arraysandfunctions are all reference types.

Static Type vs Dynamic Type

  • JavaScript is a dynamically-type languages, we do not explicitly define the types for our variables.
  • TypeScript is a superset of JavaScript, which allows static-typing. This can make your code more verbose and less prone to errors.

Primitve Typs vs Reference Typs

  • Primitive Typs - stored directly in the 'stack'
  • Reference Typs - stored in the heap
